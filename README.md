<p>A remake of the game Tetris with C++ and the raylib library</p>

## Build (Cmake)
### Windows
#### 1. Generate the Visual Studio Project

From project root:
```
cmake -B build -G "Visual Studio 17 2022"
```
#### 2. Open the Solution
```
build/TetrisRaylib.sln
```
#### 3. Build the project and run
```
Ctrl + Shift + B
F5
```
Note: All dependencies are automatically fetched via Cmake
